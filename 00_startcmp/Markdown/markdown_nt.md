## heading

* h1 -> `#` -> html `<h1>내용물</h1>`
* h2 -> `##`
* h3 -> `###`
* h4 -> `####`
* h5 -> `#####`
* h6 -> `######`

`ctrl` + `숫자` 로 전환 가능



## list 

### unorderd list (순서가 없는 리스트)

`*` / `-`

* a
  * b
    * c
      * d
        * e
          * f

`<ul></ul>`

### ordered list (순서가 있는 리스트)

`숫자` + `.` 

1. 순서가 있는 리스트 1
2. 순서가 있는 리스트2
   1. 순서가 있는 리스트 2의 1
      1. 순서가 있는 리스트 2의 1의 1

`<ol></ol>`
